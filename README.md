# Corporate Profitability & Sales Performance Data Pipeline

An end-to-end data analytics and business intelligence project built using **Alteryx** to transform raw transactional data into actionable executive insights.

## Project Architecture
The project is modularly structured into 4 sequential pipelines to ensure scalability and ease of debugging:
1. **Data Cleaning & Validation:** Handling missing profiles and filtering transaction anomalies.
2. **Strategic Calculations:** Developing robust logic for Net Profit and Margin calculations, including dynamic error-handling to prevent Divide-by-Zero execution faults.
3. **Aggregations & Grouping:** Segmenting performance by sales sectors, industries, and personnel.
4. **Data Visualization & Reporting:** Generating production-ready tables and analytical charts.

## Key Executive Insights Included
* **Top Performance Driver:** Identified the Wholesale sector (specifically Pharma & Biotech) as the primary revenue engine (~$4.25M profit at 35.1% margin).
* **Operational Optimization:** Isolated low-margin product categories (e.g., Vegetables at ~15% margin) acting as value-burners for the business.
* **Sales Variance:** Mapped high-variance performance gaps between top-performing account managers and those requiring targeted alignment programs.
